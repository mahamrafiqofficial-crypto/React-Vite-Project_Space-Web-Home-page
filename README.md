# EcoBazar – Full Stack E-commerce Web App

A full stack e-commerce web application developed using the MERN stack. This project includes product listing, product detail pages, user authentication, and RESTful APIs.

## 🚀 Features
- Product listing and product detail pages
- User authentication (Login & Register)
- RESTful API integration
- Responsive UI design using React
- Backend built with Node.js and Express
- Database management with MongoDB

## 🛠️ Technologies Used
- Frontend: React (Vite), HTML, CSS, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- API: Custom REST APIs

## 📌 Project Highlights
- Built full stack architecture from scratch
- Integrated frontend with backend APIs
- Implemented authentication system
- Hands-on experience with real-world development workflow
